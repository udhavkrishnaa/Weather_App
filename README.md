# 🌤 Weather Search App

A simple React application that allows users to search for a city and view its current weather using the Open-Meteo API.

## Features

- Search weather by city name
- Get city coordinates using Geocoding API
- Fetch current weather data
- Display:
  - Temperature
  - Wind Speed
  - Weather Condition
  - Observation Time
- Loading and error handling

## Technologies Used

- React.js
- JavaScript
- CSS
- Open-Meteo API

## How It Works

1. User enters a city name.
2. Geocoding API retrieves latitude and longitude.
3. Weather API fetches current weather data.
4. Results are displayed on the screen.
